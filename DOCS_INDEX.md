## 🎯 Quick Links

### Setup & Run
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Start: `npm start`

### Important Files
```
src/pages/index.tsx              → Main page
src/components/Hero.tsx          → Hero section
src/components/About.tsx         → About section
src/components/Projects.tsx      → Projects section
src/styles/globals.css           → Global styles & colors
public/                          → Static assets
```

### Kustomisasi Cepat
```
Warna:    src/styles/globals.css (--accent)
Konten:   src/components/*.tsx
Gambar:   public/*.jpg
CV:       public/cv.pdf
```
